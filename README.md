ムビマス 補完計画
=================
劇場版 THE IDOL M@STER MOVIE 輝きの向こう側へ　のテキスト版です。
メモ、記憶、ニコニコ動画、2chから集めてテキストにしました。
ほとんどがニコニコ動画、一部は私のメモと記憶です。
まだまだ補完が完璧ではないので、ここは違う！というところがあれば、修正お願います。
